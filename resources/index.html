Xilinx 开发工具（在用）

ISE Design Suite: System Edition

Xilinx 6 系列（Spartan-6、 Virtex-6）之前的 FPGA 器件开发，只能使用 ISE Design Suite 工具。System Edition 功能最全，包含 DSP 开发的高级工具 System Generator。

参考：ISE 官方说明

Win 10 虚拟机版，放弃

ISE 最后一个版本为 14.7，发布于2013年10月份。

2022 年 2 月，Xilinx 官网又出现了所谓的 ISE Design Suite for Windows 10 - 14.7 ，实际上是在 Win10 系统上安装虚拟机 Linux 操作系统，最终运行运行 Linux 版的 ISE 14.7。

ug1227: Provides an overview of using the ISE 14.7 Virtual Machine for Windows 10.

虚拟机版本有很多限制：最要命者是不支持使用 System Generator。
所以放弃。

Win 10 直接安装 ISE 14.7 64-bit

经过长期的使用验证，Win10 系统上可以稳定运行 ISE 14.7，配合 Matlab 2013b，利用 Sysgen 14.7 开发相应 DSP 模块亦无问题。

下载地址： 基于 Windows 7/XP/Server 的完整安装程式 (TAR/GZIP - 6.18 GB)

ISE 14.7 安装在 Win 10 系统上，参考 Xilinx 官方说明文档： 62380 - ISE Install - Installing and Running ISE 10.1 or 14.7 on a Windows 8.1 or Windows 10 machine

关键：ISE 14.7 64-bit - Turning off SmartHeap

  1. Navigate to the following ISE install directory: \14.7_DS 
  2. Rename the file "libPortability.dll" to "libPortability.dll.orig".
  3. Copy the "libPortabilityNOSH.dll" file to the same folder, and rename it to "libPortability.dll".
  4. Repeat steps 1-3 in the following folder: \14.7_DS 

The above steps substitute the original "libPortability.dll" with a "libPortability.dll" file that has SmartHeap disabled, the NOSmartHeap (NOSH) version.

ISE 14.7 的 sysgen 支持的 Matlab 最新版本为 R2013b

Vivado Design Suite

Xilinx Vivado Design Suite ，是针对 Virtex-7、Kintex-7、Artix-7、和 Zynq-7000 起的全新设计。

目前采用的版本是 Vivado 2019.1，发布时间：2019-5-29。

UG973 (v2019.1) June 7, 2019：Release Notes, Installation, and Licensing
Vivado 的新增功能(v2019.1)

2019.1 有3个更新：2019.1.1、 2019.1.2、2019.1.3，仅增加了对一些不常用新量产器件的支持；无需更新。
2019.1 的 sysgen 支持的 MATLAB 版本：R2018a、R2018b 和 R2019。

Vivado 2019.2 有较大更新：

  1. 引入 Xilinx 统一安装程序（Xilinx Unified Installer）。单个统一安装程序有助于用户安装所有 Xilinx 工具，如 Vitis、Vivado、On-premise Install for Cloud Deployment（支持云部署的本地安装）、Bootgen、Lab Edition、Hardware Server 以及 DocNav（文档导航器）等。
  2. Vivado System Edition 摒弃了 Xilinx SDK （继承自 ISE 的嵌入式软件开发环境），而全面用 Vitis 代替之。

Vitis 替代 SDK 的跳跃是目前没有升级 Vivado 版本的主要原因。

Vivado各版本下载官网地址。
